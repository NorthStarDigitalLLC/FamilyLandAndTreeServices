# GitHub Pages Website

A fully functional static website template that works with GitHub Pages.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Features

- Responsive layout
- Dark mode toggle with localStorage
- Mobile navigation
- Smooth scrolling
- Client-side contact form validation

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload all files to the repository root.
3. Go to **Settings** -> **Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or `master`) and `/root`
5. Save the settings.
6. Wait for GitHub Pages to publish your site.

## Customize

- Update the brand name in `index.html`
- Change colors in `styles.css`
- Edit content sections and links
- Replace the contact form behavior in `script.js` with your preferred backend or service
