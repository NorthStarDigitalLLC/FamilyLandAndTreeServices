# Hot Headz GitHub Pages export

## What is included
- `/index.html` home page
- `/menu/index.html` menu page using your provided menu code
- `/menu-data/index.html` sample JSON data source for the menu page

## Important note
Only one uploaded source file had content: `Menu Code.txt`.
The other uploaded `.txt` files in the zip were empty, so this package uses a best-effort static home page plus your real menu code.

## GitHub Pages steps
1. Upload all files from this folder to a GitHub repository.
2. Enable GitHub Pages from the main branch.
3. If your repo is a project site instead of a user site, absolute paths like `/menu/` and `/menu-data/` may need to be changed to relative paths.
4. Update `/menu-data/index.html` with your real menu JSON.
